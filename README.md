一．用户登录  
1.完整的登录，注册功能（采用Property类保存）  
2.用户名已存在不可注册  
3.下一次运行依然可以登陆  
4.游客模式，默认名称Tourist，不可存读档  
5.点选“记住我”按钮可记住当前用户，下一次直接登录  
6.重置输入快捷按钮  
  
二．主界面  
1.界面，按钮，UI全部重置（JavaFX）带有动画  
2.界面可全屏，窗口是兼容的  
3.游戏有BGM  
4.提供不同头像  
5.两种模式：人机对战&联网双人对战  
  
三．游戏功能
1.全部棋子的移动规则和吃子逻辑（蹩马腿，蹩象眼，将不对将，卒过河可横着走，象不过河等等）  
2.可选边（人机对战电脑同步到另一侧）  
3.悔棋（人机对战悔棋电脑同步）  
4.记谱（按照中国象棋记谱规则）  
5.存读档  
6.认输  
7.认输后可返回主界面，可自由选择新的模式和新的先手方  
8.认输后可复盘  
  
四．功能细化-棋盘逻辑  
1.走棋有动画（线程 Thread）  
2.走棋有提示位置和吃子位置  
3.重做了全部棋子及棋盘  
4.将军高亮提示  
5.将死逻辑的全部实现（暂无困毙逻辑）但对困毙下一方走后可以吃掉其将/帅来达成胜利  
6.将死后弹窗提示游戏结束  
  
五．功能细化-存读档  
1.只能登录用户实现存读档  
2.用户只能读自己的存档  
3.存档可以覆盖  
4.坏档会弹窗提示但程序运行不报错  
5.存档文件以.xiangqi后缀保存  
6.可以选择存档存储的位置  
7.存读档兼容记谱功能  
8.存读档兼容悔棋  
9.存读档兼容复盘  
10.存读档兼容人机对战  
  
六．功能细化-人机对战  
1.AI核心（Zobrist哈希编码，多个方法评估棋盘局势，每个棋子具有自己的value，不同棋局阶段具有不同的行为模式，静态搜索(quiescence search)，贪心，Alpha beta剪枝）AI从数字棋盘类SimBoard同步  
2.AI在超过计算时限Time Limit后会采用当前计算出最优走法（Time Limit可自行设置）  
3.AI搭载约40000盘棋谱，可根据对手行为走出谱棋  
4.AI兼容悔棋  
5.AI兼容存读档  
6.AI兼容记谱  
  
七．功能细化-联网双人对战  
1.邀请方为红方    
2.双方轮流走棋  
3.记谱功能兼容联网对战  
4.实现游戏大厅的登录功能，退出登录功能  
5.服务器端有完整的控制台UI，交互和显示  
6.联网对战兼容记谱  
7.联网对战兼容复盘  
8.联网对战兼容悔棋  
  
八．功能细化-复盘  
1.棋局结束后可复盘当前棋局  
2.五个按钮：开局，上一步，播放，下一步，终局  
3.播放每间隔2s走一步，走完自动停止播放  
4.复盘兼容存读档
